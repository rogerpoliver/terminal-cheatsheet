# Kitty Cheatsheat

### Split panel
Ctrl + Shift + Enter

### Move splitted panel
Ctrl + Shift + F ou Ctrl + Shift + B

### Select next splitted panel:
Ctrl + Shift + ] ou Ctrl + Shift + [

### Close splitted panel:
Ctrl + D

### Change panel layout:
Ctrl + Shift + L

### Change splitted panel size:
Ctrl + Shift + R

### Select emojis:
Ctrl + Shift + U

### Rename tab:
Ctrl + Shift + Option + T
